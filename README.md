# DEE on macOS Notes

This repository is for documentation and reproducible notes about running Dolby Encoding Engine workflows on macOS with `gcenx/wine`.

## Included

- Markdown documentation
- Lightweight text notes

## Excluded

- Dolby tool binaries/packages (`.exe`, `.dll`, `.zip`)
- Licenses (`.lic`)
- Media test assets (`.wav`, `.ec3`)
- Runtime logs (`.log`)
- Extracted engine folders

Primary guide:

- `DEE_Encoding_on_macOS_with_gcenx_wine.md`
